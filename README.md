# 记忆代理人 - 传说武器锻造台

这是为「记忆代理人 x 落花星雨」服务器制作的专属武器定制网页。玩家可以在这里可视化设计自己的传说武器，系统会自动计算价格并生成配置代码。

## 🎮 玩家使用指南

1. **填写信息**：输入你的游戏 ID，选择想要的武器材质（如钻石剑、下界合金斧等）和自定义名称。
2. **定制属性**：
   - 在左侧选择需要的属性（攻击、防御、特殊效果等）。
   - **计费公式**：
     > 系统采用动态惩罚算法，防止数值膨胀。
     
     **最终价格计算：**
     ```
     最终价格 = max(50, ∑(单项价格 × 部位倍率))
     ```
     *(即：总价最低 50 元起步，所有属性价格加总后取整)*

     **单项价格计算逻辑**：
     1. **基础消耗** = 属性数值 × 单价系数
     2. **超限惩罚** (当数值 > 软上限时触发)：
        ```
        单项价格 = 基础消耗 × (1 + 2 × (数值 - 软上限) / 软上限)
        ```
        *(解析：超出软上限的比例每增加 100%，该属性价格在基础之上额外翻 2 倍)*
     
     **参数示例**：
     - **部位倍率**：副手 = 2.0，其他部位 = 1.0
     - **最低消费**：50 元
3. **提交订单**：
   - 满意后点击右侧的 **「复制配置代码」** 按钮。
   - 将复制的内容直接发送给服主进行实装。
